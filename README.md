## 项目启动

### 文件存储路径
修改`settings-development.json`中的`storagePath`为自己的任意目录，注意权限问题。

在development环境下，设置为**用户目录**（ ~/ ）中任意目录，以保证有读写权限。

```js
npm i
meteor
```
